installation:
download dependencies.zip, and extract the files to game-folder in "X:\Program Files\Steam\steamapps\common\Left 4 Dead 2"
then delete the file "x:\Program Files (x86)\Steam\steamapps\common\Left 4 Dead 2\bin\libcef.dll"
https://www.mediafire.com/file/o9nga1scczjs62p/dependencies.zip/file

If you experience unexpected errors, Follow the steps below
1. Remove launch commands or configs
2. Re-install Directx and Visual C++ Redistributable 2005-2019
3. Dont run left4dead2.exe in compatibility mode
4. Uninstall Reshade/ENB or any d3d9 dll based mods
5. Disable all workshop addons
6. Disable anti virus softwares
7. Disable Nvidia/AMD upscalers or enhancers
8. Make sure dependable files are extracted to game folder
